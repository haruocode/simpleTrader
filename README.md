# simpleTrader

シンプルな仮想通貨自動売買プログラム

## 事前に必要なもの

- Node.js
- Bitflyer口座のAPIキー(本番稼働する場合)
- お金(本番稼働する場合)

## アルゴリズム

- SMA(期間20)がSMA(期間100)より上＆現在価格(1本前のローソクの終値)がSMA(期間20)の下から上に突き抜けた場合、成行買い。
- SMA(期間20)がSMA(期間100)より下＆現在価格(1本前のローソクの終値)がSMA(期間20)の上から下に突き抜けた場合、成行売り。
- 途転取引はしない。

## 実行方法
### 開発

```
npm run dev
```

### 本番用ビルド
```
npm run build
```
